Classifying, concatenating, and shuffling¶

Now that the data is read in and appears okay. However it is still split into two distinct files. Whereas they should be as one.

  1. Merge the two data sets.
  2. Add classification labels to both dataframes (Fake is 1, True is 0)
  3. Concatenate the dataframes
  4. Shuffle the data to avoid biasing while splitting data later. This is especially important since stratifying is not readily doable with textual data.

def prep_df(df_fake, df_true):
    '''
    Takes two input dataframes for fake and true articles
    Assigns a boolean class to each (a fake article is positive and gets value 1)
    Then concatenates into one dataframe, and shuffles at random
    '''
    
    df_fake["isFake"] = 1
    df_true["isFake"] = 0
    
    df_out = (pd
              .concat((df_fake.dropna(axis = 0, how = 'any'), df_true.dropna(axis = 0, how = 'any')))
              .sample(frac = 1, replace = False, random_state = 200))
    
    '''Note: pd.sample with frac = 1 and replace = False is effectively a random shuffle'''
    
    return df_out

A sequence of functions for formatting text, removing stop words, and stemming¶

Before proceeding, we need to make sure that the text above (which is easily readable by a human), needs to become machine-readable. That means breaking up a sentence into words, and identifying words as indices.

I will eventually get into indexing words, but for now, the following pre-processing is done:

  1. First, remove any HTML tags (there aren't any in the text, but this is to future-proof any implementation).
  2. Turn all letters into lowercase.
  3. Split strings into lists of words.
  4. Remove all stop words from the lists.
  5. Use PortStemmer() to stem words to their shortest distinct form.

def format_clean(article):
    from bs4 import BeautifulSoup
    import re
    
    '''
    Removes HTML tags and changes to lower case.
    Then splits text into a list of individual words.
    '''
    
    text = BeautifulSoup(article, "html.parser").get_text()
    text = re.sub(r"[^a-zA-Z0-9]", " ", text.lower()) 
    
    return text.split()

def remove_stopwords(text):
    import nltk
    from nltk.corpus import stopwords
    
    '''
    Remove stop words (e.g. the, a, at, an)
    Return list of words without stop words
    '''
    
    nltk.download("stopwords", quiet = True)
    
    words      = []
    for word in text:
        if word not in stopwords.words("english"):
            words.append(word)
            
    return words

def stem_words(text):
    from nltk.stem.porter import PorterStemmer
    
    '''
    Stem each word to its least ambiguous form (e.g. president --> presid)
    To account for tenses and so on.
    Returns a list of stemmed words
    '''
    
    stemmer = PorterStemmer()
    words   = []
    for word in text:
        words.append(stemmer.stem(word))
        
    return words

The upcoming functions will apply the preceding sequence of functions and create a pickle file for easier loading later.¶

The following function will process the above functions en masse for all articles in the data set. It will then save the result as an output .pkl file.

That pickle file is especially important since each run can take well upwards of an hour.

If a pickle file already exists, it will simply read it in, saving substantial compute (and actual) time.

def pickle_dump_words(X_train, X_test, y_train, y_test, cache_dir, cache_file):
    '''
    Performs the word cleaning and stemming function on all the articles in the dataset
    Places each article into a list of articles
    Then outputs a dictionary of the cleaned lists and the corresponding labels
    Dumps into a pickle file for easy loading later.
    '''
    
    words_train = [article_to_words_clean(article) for article in X_train]
    words_test  = [article_to_words_clean(article) for article in X_test]
    
    cache_data  = dict(words_train = words_train, words_test = words_test,
                       labels_train = y_train, labels_test = y_test)
    
    file        = open(os.path.join(cache_dir, cache_file), 'wb')
    
    
    pickle.dump(cache_data, file)
    print('File created!')
    
    return cache_data

def create_cache_file(cache_dir, cache_file): 
    '''
    Looks for a previously created pickle file to load
    If none found, pass None
    Overrides errors for no file found using a try/except
    '''
    try:
        '''First look for previously loaded in cache files'''
        file       = open(os.path.join(cache_dir, cache_file), 'rb')
        cache_data = pickle.load(file)
        print("Read data from cache file:", file)
        
    except:
        '''If it's not found, override the error using a try/except and proceed to creating file'''
        print("File not found, producing file.")
        cache_data = None
        pass
    
    return cache_data

Word dictionary¶

Now for the first step in indexing words...

The following function will now sift through all the data, find the $N-2$ most common words, and order them by frequency.

$N = 5000$ by default.

The reason it looks for $N-2$ words and not $N$ is because it saves 2 extra indices: one for "no word" (index 0) and one for infrequent words (1). The most frequent word gets the next index (2), and so on.

The function also separately outputs the number of times each word appears. This is not for the model. This is purely for analytical purposes; so I can visualize which words are most common for true & fake data.

In [23]:
def build_and_save_dict(data, vocab_size = 5000):
    '''
    Creating a dictionary of words based on frequency of appearance
    The function also allows for no word labels (index = 0) and 'infrequent' words (index = 1)
    The no word and infrequent labels are assigned in a later function
    
    INPUT:
    * data: a list of sentences
    * vocab_size: the desired output size of the dictionary. Function will find the most common
                  (vocab_size - 2) words. The other 2 indices are for no words or infrequent words.
                  
    RETURNS:
    * word_dict: a dictionary of most common words ordered from most to least frequent. The indices
                 correspond to the word frequency rank, not the actual frequency of appearance. 
                 THIS IS WHAT IS USED AS INPUT TO THE MODEL.
    * word_count: a dictionary of words with the frequency of appearance. This is NOT used by the model,
                  but is simply collected for analytical purposes.
    '''

    word_count = {} 
    for sentence in data:
        for word in sentence:
            if word in word_count:
                word_count[word] += 1
            else:
                word_count[word] = 1
                
    sorted_words = sorted(word_count, key = word_count.get, reverse = True)
    
    word_dict = {} 
    for idx, word in enumerate(sorted_words[:vocab_size - 2]): 
        word_dict[word] = idx + 2  
        
    with open(os.path.join('pytorch_files/word_dict.pkl'), "wb") as f:
        '''
        No need to check if old file exists. Process is fast enough that can just overwrite every time.
        Also overcomes added complexities if need to later change vocab_sizen in a given run
        '''
        pickle.dump(word_dict, f)
        
    return word_dict, word_count

And now to convert all words into indices¶

The following function will:

  1. Take the full data set.
  2. Read in each sentence (already stemmed and processed).
  3. Assign each word an index based on the above produced word_dict
  4. Pad/truncate each sentence to keep them all the same length (avoid biasing model by length).
  5. Output a machine-readable form of the data.

def verbose_to_indices_all(word_dict, data, pad = 500):
    ###adapted from first project 
    
    ''' 
    Takes in all the verbose reviews in the training data and returns reviews comprised of of indices, 
    based on word locations in word_dict.
    Assign the first two indices (0,1) to no words, or infrequent words, respectively. 
    Starts by assigning noword to all word locations, then add words one at a time
    
    Input arguments:
    * word_dict: a dictionary object with words ordered by frequency, from most to least frequent
    * data: a list of sentences, NOT an array or other type of object
    * pad: the maximum allowed length of a sentence (default: 500). If a sentence is shorter than 500 words
           it will pad it to 500. A sentence longer than 500 words will be truncated to 500. 
    
    Returns:
    A numpy array of the the processed data in indices, concatenated with an array of the sentence lengths.
    '''
    noword = 0 
    infreq = 1 
    
    data_out = []
    data_len = []
    
    for sentence in data:
        
        sentence_pad = [noword] * pad
        
        for ind, word in enumerate(sentence[:pad]):
            if word in word_dict:
                ###if word is in the dictionary, assign its corresponding index
                sentence_pad[ind] = word_dict[word]
            else:
                ###else just fill it with the infrequent index (1)
                sentence_pad[ind] = infreq
    
        data_out.append(sentence_pad)
        data_len.append(min(len(sentence), pad))
        
    return np.array(data_out), np.array(data_len)

def verbose_to_indices_single(word_dict, sentence, pad=500):
    ''' 
    This is similar to teh verbose_to_indices_all() function above, but designed for one single sentence
    That way it can be used in a for loop on a predictor object to test on external data sources
    '''
    noword = 0 
    infreq = 1 
    
    data_out = []
    data_len = []
        
    sentence_pad = [noword] * pad

    for ind, word in enumerate(sentence[:pad]):
        if word in word_dict:
            ###if word is in the dictionary, assign its corresponding index
            sentence_pad[ind] = word_dict[word]
        else:
            ###else just fill it with the infrequent index (1)
            sentence_pad[ind] = infreq

    data_out.append(sentence_pad)
    data_len.append(min(len(sentence), pad))
        
    return np.array(data_out), np.array(data_len)
